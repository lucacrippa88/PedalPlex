Keeley Remote Switcher (Dream)





The Keeley Remote Switcher for the Dream pedal is an accessory designed to provide convenient external control for Keeley’s Dream effects unit. By connecting the remote switch, players can toggle functions or activate preset features without needing to reach the pedal itself. This is particularly useful during live performances where quick changes are required. The remote switch integrates seamlessly with compatible Keeley pedals and features rugged construction for stage reliability. Its compact design allows easy placement on pedalboards. For players using Keeley’s Dream series pedals, the Remote Switcher provides enhanced performance control and convenience.
